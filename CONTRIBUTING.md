# Cómo Contribuir

¡Gracias por tu interés en mejorar Clara 4.0!

## Formas de Contribuir

1. **Comparte tu caso de uso** - Abre un Issue contando tu experiencia
2. **Reporta problemas** - Si algo no funciona, cuéntanos
3. **Sugiere mejoras** - Ideas para nuevas funcionalidades
4. **Mejora la documentación** - Corrige errores o añade ejemplos

## Proceso Simple

1. Para sugerencias/bugs: Abre un Issue
2. Para cambios en documentación: 
   - Haz un Fork
   - Edita el archivo
   - Envía Pull Request con descripción clara

## Lo Que NO Aceptamos

❌ Spam o contenido promocional  
❌ Contenido ofensivo  
❌ Cambios que rompan la funcionalidad  

---

¿Dudas? Abre un Issue con etiqueta "question"
